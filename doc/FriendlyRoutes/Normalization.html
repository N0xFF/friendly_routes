FriendlyRoutes Normalization

Pass normalized options to url_for().

url_for() uses ActionDispatch::Routing::RouteSet::Generator and so we added additionally normalization to initialize method of this class.

If friendly_route present we take friendly params from recall to options and remove corresponding non-frigidly params

def initialize(*options)
  super

  return unless friendly_route = @recall[:friendly_route]

  friendly_route.params.each do |param|
    param          = param.name
    prefix         = friendly_route.prefix
    prefixed_param = FriendlyRoutes::PrefixedParam.new(param, prefix)

    use_recall_for(prefixed_param.call)
    @options.delete(param)
  end
end
